## Nape Notify v0.1.4

Windows 11向けのポータブル配布版です。

### Highlights

- インストール不要
- 管理者権限不要
- Keychron Launcher DOM監視に対応
- Chrome DevTools Protocolが使えない場合はMock providerへフォールバック
- レイヤー/角度変更時の半透明オーバーレイ通知
- タスクトレイ常駐

### Downloads

- `NapeNotify-0.1.4-portable-exe-only.zip`
  - 展開して `Nape Notify.exe` を起動してください。
- `NapeNotify-0.1.4-win-unpacked.7z`
  - サイズ優先版です。展開後、フォルダ内の `Nape Notify.exe` を起動してください。

### Notes

- 未署名アプリのため、Windows SmartScreenの警告が表示される場合があります。
- Keychron LauncherをChrome上で開き、Nape Proと接続した状態で利用してください。
